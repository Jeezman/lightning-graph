# lightning-graph

A Lightning graph visualizer

## Technologies Used

 - NodeJS/Typescript (Backend)
 - React/Typescript (Frontend)
 - D3 
 - Lightning Polar (LND)

## How To Run

Clone the Github repository 

```git clone https://github.com/Jeezman/lightning-graph.git```
